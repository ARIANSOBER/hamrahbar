# همراه بار — پلتفرم هوشمند لجستیک

یک پلتفرم مدرن، حرفه‌ای و کاملاً ریسپانسیو برای مدیریت حمل و نقل و لجستیک.

## تکنولوژی‌ها

- **Next.js 16** — فریم‌ورک React با App Router
- **TypeScript** — تایپ‌سیف برای کل پروژه
- **Tailwind CSS v4** — سیستم استایل ماژولار و سریع
- **Framer Motion** — انیمیشن‌های حرفه‌ای
- **Lucide Icons** — آیکون‌های با کیفیت

## نحوه اجرا

```bash
cd hamrahbar
npm run dev
```

سایت روی آدرس `http://localhost:3000` در دسترس خواهد بود.

## فایل‌های ترجمه

فایل‌های ترجمه در مسیر `src/data/translations/` قرار دارند:

- `fa.json` — ترجمه فارسی (پیش‌فرض)
- `en.json` — ترجمه انگلیسی

برای اضافه کردن یک کلید ترجمه جدید، آن را به هر دو فایل اضافه کنید.

## دیتاهای آزمایشی

تمامی دیتاهای استفاده شده در پروژه در پوشه `src/data/` قرار دارند. این دیتاها دقیقاً به همان شکلی که APIهای واقعی قرار است برگردانند، ساختاربندی شده‌اند. برای اتصال به بک‌اند واقعی:

1. تابع‌های mock در فایل‌های `/data` را با فراخوانی API جایگزین کنید
2. تایپ‌های موجود در `src/types/index.ts` منطبق با پاسخ API هستند
3. فایل `server.js` در مسیر `redesign/` قبلی یک نمونه بک‌اند ساده دارد

## طراحی

- **زبان پیش‌فرض:** فارسی
- **جهت:** راست‌به‌چپ (RTL)
- **پشتیبانی از:** حالت تاریک و روشن
- **کاملاً ریسپانسیو:** موبایل، تبلت، دسکتاپ
- **دسترسی:** کیبورد ناوبری، ARIA، فوکس

## تم

کاربر می‌تواند بین حالت تاریک (پیش‌فرض) و روشن جابجا شود. تنظیمات در `localStorage` ذخیره می‌شود.
